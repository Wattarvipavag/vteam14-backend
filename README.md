# Backend repo for vteam14

#### **Clone the repository**

```bash
git clone git@github.com:jkberiksson/vteam14-backend.git
```

#### **Change directory**

```bash
cd vteam14-backend
```

#### **Install dependencies**

```bash
npm install
```

---

## **Docker environment**

#### **Run the application**

```bash
docker compose up -d
```

#### **Stop the application**

```bash
docker compose down
```

---

## **Local environment**

#### **Run the application**

```bash
npm run dev
```
