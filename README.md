# Backend repo for vteam14

#### **Clone the repository**

```bash
git clone git@github.com:jkberiksson/vteam14-backend.git
```

#### **Change directory**

```bash
cd vteam14-backend
```

#### **Install dependencies**

```bash
npm install
```

---

#### **Run the application in a docker container**

```bash
docker compose up -d
```

```bash
docker compose down
```

---

#### **Run the application locally**

```bash
npm run dev
```
